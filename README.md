# BTT-Mod

A modification for Super Mario Odyssey made to make grindin BTT segmnts easier.

## Controls
L-Stick -> Toggle Menu  
L + DPad-Left -> Toggle Menu Input
DPad -> Move Selection  
A -> Activate Selection
